CDN Usage Example

Welcome to the CDN Demo

This page demonstrates how to use assets hosted on your CDN.

The stylesheet and JavaScript library are loaded from the CDN, providing:

  • Faster load times through caching
  • Reduced server load
  • Better performance for users worldwide
  • Easy version management
View Asset Catalog

How to Use

1. Include the CSS in your HTML head:

<link rel="stylesheet" href="https://your-cdn-domain.com/css/style.css">

2. Include the JavaScript before closing body tag:

<script src="https://your-cdn-domain.com/js/main.js"></script>

3. Use CDN utilities in your code:

CDN.loadScript('path/to/script.js');
CDN.loadStylesheet('path/to/style.css');
CDN.fetchJSON('path/to/data.json').then(data => console.log(data));
